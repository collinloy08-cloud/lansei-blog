# 岚生的个人博客

这是一个纯静态、双语个人博客系统，不需要数据库或后端。直接打开下面两个文件即可使用：

- `index.html`：博客前台
- `editor.html`：可视化内容管理器

## 已具备功能

- 首页最近文章流，不使用归档页当首页
- 精选文章、全部文章、文章详情页
- 年份归档、分类页、标签页
- 中文 / English 双语切换
- 文章详情页自动生成左侧文章目录抽屉，支持标题编号和当前位置高亮
- 阅读进度条、阅读百分比、回到顶部按钮
- 滚动入场、卡片悬停、吸附侧边栏等轻量动态效果
- 全文搜索，包含标题、摘要、正文和标签
- 明暗主题切换
- 侧边栏作者信息、统计、最近更新、分类、站点概览和链接
- 可视化编辑器，不需要手改代码

## 怎么修改内容

打开 `editor.html`，顶部先选择编辑语言：

- 选择 `中文`：修改中文站点文案、中文文章标题、中文摘要和中文正文
- 选择 `English`：修改英文站点文案、英文文章标题、英文摘要和英文正文

默认开启了“中文自动同步英文”。你编辑中文字段后，编辑器会自动更新对应英文内容；也可以点“同步当前内容到英文”手动同步当前页签。翻译会优先使用浏览器翻译能力，其次尝试在线翻译接口；如果翻译不可用，会生成带 `Needs English polish` 标记的英文草稿，提醒你稍后润色。

这些内容会保存在同一个 `content.js` 里。Slug、日期、分类、标签、是否精选、是否草稿只需要维护一次。

编辑器里可以修改：

- 博客名称、首页标题、首页介绍
- 作者介绍
- 站点概览
- 外部链接
- 分类
- 文章标题、日期、分类、摘要、标签、正文
- 文章是否精选、是否草稿

编辑完成后，点击 **保存 content.js**。如果浏览器不支持直接保存，编辑器会自动下载新的 `content.js`。

## 写文章

文章正文支持简单 Markdown：

- `## 小标题`
- `### 更小的标题`
- `- 列表项`
- `> 引用`
- 普通段落

文章页会根据 `##` 和 `###` 自动生成侧边目录。

已有两个模板可以参考：

- `templates/research-note.md`：科研 / 论文笔记模板
- `templates/essay-note.md`：散文 / 纪事模板

## 预览英文版

在前台点击右上角 `EN` 即可切换英文版。也可以打开：

`index.html?lang=en`

## 文件说明

- `content.js`：所有博客内容数据
- `script.js`：博客前台交互
- `styles.css`：博客样式
- `editor.html`：内容管理器页面
- `editor.js`：内容管理器逻辑
- `editor.css`：内容管理器样式

## 重要提示

如果你在 `editor.html` 保存后，`index.html` 没有变化，刷新一下页面。浏览器可能缓存了旧的 `content.js`。
